Built to get out of your way.

The one interaction that matters — hold the modifier, tap to cycle, release to switch — with everything you need to land on the right window fast.

▤

Live preview

See the actual window next to the list, not just an app icon — so you land on the right one the first time.

↺

Most-recent order

Windows are ordered by when you last used them, so the one you want is almost always second in the list — one tap away.

⌨

Keyboard or mouse

Hold-and-tap to cycle and release to confirm, or hover the list and click. Hovering previews; leaving the list snaps back to your selection.

⤢

Every window

Switches between individual windows, not just apps — including multiple windows of the same program.

One design, native on each.

Not a cross-platform wrapper — each version is a native app that shares the same behavior and switching model.

macOS

Swift · AppKit

A menu-bar app that takes over ⌘Tab / ⌘` with four hotkey slots, an optional 3-finger trackpad gesture, and live window thumbnails.

Windows

C# · .NET · WPF

A tray app that replaces Alt+Tab, with live DWM window previews composited right into the switcher and mouse hover + click support.
